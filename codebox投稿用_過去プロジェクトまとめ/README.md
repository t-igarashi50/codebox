# codebox投稿用 過去プロジェクトまとめ

AIで作った過去プロジェクトを、codebox投稿テスト用にまとめたZIPです。

## 投稿タイトル案

AIで作った過去プロジェクトまとめ

## コメント案

これまでAIで作ったLP、Webサイト、音声編集ツール、デザイン補助ツールなどをまとめました。完成品も実験中のものも混ざっています。

## 中身

- `index.html`: ブラウザで開ける見本ページ
- `projects.md`: 過去プロジェクト一覧
- `sample-code.js`: codeboxでコード表示を試すための短いサンプル

## codeboxでの使い方

1. このフォルダをZIPにする
2. codeboxの投稿画面で「作品ZIP」にドロップ
3. タイトルと削除用パスワードを入れる
4. 投稿

このZIPは投稿テスト用なので、重い完成品ファイルは入れていません。
